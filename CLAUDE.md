# 고구마마켓 개발 가이드

## 프로젝트 개요
- 이 폴더는 개발 연습용 프로젝트입니다
- **당근마켓과 유사한 중고거래 사이트 "고구마마켓"** 을 단계적으로 만들어가고 있습니다
- `goguma-market/` 폴더가 Next.js 앱 본체입니다
- `index.html` 은 별개의 가계부 앱입니다 (같은 Supabase DB 공유)

## 기술 스택
- **Next.js** (화면을 만드는 프레임워크)
- **Supabase** (데이터베이스 + 로그인 기능)
- **Vercel** (인터넷에 배포하는 서비스)
- **Tailwind CSS** (디자인 스타일)

## Supabase 프로젝트
- Project ID: `ojydecoyvqbrhqrevmso`
- 가계부(index.html)와 고구마마켓이 같은 DB를 사용합니다

## Claude 행동 지침

### 설명 방식
- 개발 용어는 비개발자도 이해할 수 있게 쉽게 설명한다
- 예) "Route Handler" → "서버가 요청을 받아 처리하는 코드"
- 예) "미들웨어" → "모든 페이지 이동 전에 자동으로 실행되는 검사 코드"

### 작업 방식
- 사용자가 지시한 것만 단계적으로 수행한다 (추가 기능 임의 구현 금지)
- 작업 중 무엇을 하고 있는지 짧게 자주 알려준다
- 코드 수정 → 빌드 확인 → 화면 테스트 순서로 진행한다

### 사람이 직접 해야 하는 작업 안내
- Supabase 대시보드 설정 변경
- Vercel 환경변수 등록
- 이메일 인증 등 외부 서비스 설정
- 위 작업이 필요할 때는 **무엇을**, **어디서**, **어떻게** 해야 하는지 번호를 붙여 알기 쉽게 안내한다

### 테스트
- 개발이 완료되면 항상 미리보기 서버를 열어 화면을 직접 확인한다
- 주요 기능(버튼 클릭, 폼 제출 등)은 자동으로 테스트하고 결과를 보고한다
